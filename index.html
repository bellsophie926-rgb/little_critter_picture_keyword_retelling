Little Critter's Secret Treasure Hunt

看图片 + 关键词复述练习｜学生先不用看原文，用完整句把故事讲出来。

使用方法

  1. 第一遍：学生只看图片，说出人物、地点、动作。
  2. 第二遍：看关键词，用 I / We 和完整句复述每一页。
  3. 第三遍：隐藏中文提示，尽量不用看句型支架，把 8 页连成完整故事。
  4. 老师检查：最后点开“参考复述”，看学生是否漏掉核心情节。

Sentence Starters

复述要求：至少说 3 句。尽量使用关键词，但不要逐字背原文。可以加入连接词：First / Then / Next / Finally。
老师用：参考复述
